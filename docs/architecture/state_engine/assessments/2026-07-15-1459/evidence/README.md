# State Engine Assessment Evidence

This directory preserves the evidence inventory behind the 2026-07-15 token
scheduler assessment. It distinguishes executed proof from read-only readiness
analysis.

- [Wave 1 evidence](wave-1-evidence.md) records the executed packages and
  confirmed gaps.
- [Wave 2 readiness evidence](wave-2-readiness.md) records production caller
  chains, strongest current tests, and exact next discriminators.
- [Verification runs](verification-runs.md) records retained commands and the
  limits of each result.

The synthesized verdict is in
[03-executed-evidence.md](../03-executed-evidence.md). Evidence here does not
independently change a leg's verdict.
