Open source · MIT · Python 3.12+

High-assurance pipelines for decisions you have to defend.

ELSPETH is open-source software developed and publicly released by staff of the DTA — Australia's Digital Transformation Agency. It's a high-assurance pipeline substrate for consequential workflows, where a wrong output can cause operational, legal, safety, or financial harm. Author in reviewable YAML or the LLM Web Composer; both run on one audited engine.

Why ELSPETH

Substrate first. Authoring second.

A pipeline is made of declared primitives carrying schema and semantic contracts. Validation and audit are core product properties — not after-the-fact diagnostics.

Two surfaces, one engine

Hand-edit reviewable YAML, or build by conversation in the Web Composer. Both target the same runtime, contracts, and audit trail.

Explore authoring

Audit is the product

The Landscape database records full row lineage — hashes, route decisions, terminal outcomes, and provenance — as the canonical evidentiary record.

See the assurance model

For high-stakes work

Tender evaluation, document QA, content moderation, financial compliance — same framework, different plugins, full audit trail.

Install and run.

Install the CLI, or open the Web Composer and describe what you want. Both end in a pipeline that can be validated, executed, audited, and explained.
